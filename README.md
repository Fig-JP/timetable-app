# 鶴岡高専 時間割アプリ

鶴岡工業高等専門学校の授業時間割を管理するPWA（Progressive Web App）です。

> Created by [Fig](https://fig-jp.github.io/website/about)

-----

## 機能

### 時間割タブ

- 今日の時間割を縦リストで表示
- `‹ 今週 ›` ナビゲーションで先週・来週の時間割を閲覧
- 授業中・休憩中・授業前の状態をプログレスバーで表示（残り時間つき）
- セルをタップして授業の追加・編集・休講・補講・変更を設定

### 一覧タブ

- 週間時間割をグリッド表示
- `‹ 今週 ›` ナビゲーションで週を切り替え
- 「本来の時間割を表示」トグルで通常時間割と当週の変更を切り替え
- 週間表を画像として共有・保存

### 課題タブ

- 科目・期限つきで課題を登録・管理
- 期限切れ・期限間近の課題を色分け表示
- 完了チェックで課題を管理

### 出席タブ

- 科目ごとの欠席回数をカウント
- 総授業数から**欠席上限（1/4）を自動計算**
- 前期のみ・後期のみ・通年の区分を科目ごとに設定
- A/B週設定に応じて週あたりコマ数（週1 or 週2）を自動判定
- 前期・後期の授業週数を手動設定（デフォルト各15週）
- 上限に近づくと警告、超えると「アウト」表示
- 科目名をタップして詳細設定（期間・週数・上限手動上書き）

### 設定タブ

- **クラス名設定**：ヘッダーに「鶴岡高専 ◯◯」と表示
- **カレンダー**：日付ごとにA/B週・日程振替・特別ラベル（試験日など）・時間割非表示を設定
- **授業開始通知**：各コマ3分前にプッシュ通知（要許可・アプリ起動時にその日分をスケジュール登録）
- **データ管理**：JSONエクスポート・インポート・ICSエクスポート

## 授業の登録

### 隔週設定（A週・B週）

上部の `A週` / `B週` ボタンで表示週を切り替えます。  
授業登録時に「A週のみ」「B週のみ」「毎週」を選択できます。

### 授業変更・休講・補講

セルをタップ → ステータスを選択 → 対象日付と詳細を入力して保存。  
変更は指定した日付のみ適用され、通常の時間割には影響しません。  
当日以外の変更も保存され、アプリを開き直してもリセットされません。

### 日程振替

設定タブのカレンダーから日付をタップ → 「日程振替」で曜日を選択。  
例：火曜日だが月曜日程で授業を行う場合、その日の時間割・通知が月曜のものに切り替わります。

## 出席管理

科目ごとに期間（通年・前期のみ・後期のみ）と授業週数を設定すると、総授業数と欠席上限が自動計算されます。

|設定項目   |内容                       |
|-------|-------------------------|
|授業週数   |前期・後期それぞれ設定（デフォルト各15週）   |
|期間     |通年 / 前期のみ / 後期のみ         |
|週あたりコマ数|時間割の設定から自動判定（毎週→週2、隔週→週1）|
|欠席上限   |総授業数 ÷ 4（切り捨て）。手動上書きも可   |


> 前期は9月末まで。

## ホーム画面への追加

### iOS（Safari）

1. Safari でアプリを開く
1. 共有ボタン →「ホーム画面に追加」

> **注意**：通知機能はホーム画面から起動した場合のみ動作します。

### Android（Chrome）

1. Chrome でアプリを開く
1. アドレスバー右のメニュー →「ホーム画面に追加」

## 通知について

授業開始3分前にプッシュ通知を送信します。

- 設定タブ →「授業開始通知（3分前）」をタップして許可
- 通知はアプリ起動時にその日の分をスケジュール登録
- **毎朝アプリを一度開く**ことで確実に通知が届きます
- 通知内容：コマ番号・科目名・教員名・教室

> バックグラウンドでの通知はOSの制限により届かない場合があります。  
> iOSはホーム画面追加済みのPWAのみ対応、Androidは比較的安定しています。

## データの管理

### JSONエクスポート・インポート

設定タブ →「JSONでエクスポート」／「JSONをインポート」

#### JSONに含まれるデータ

|キー           |内容               |
|-------------|-----------------|
|`tt`         |時間割データ           |
|`ov`         |授業変更・休講・補講       |
|`weekMap`    |日付ごとのA/B週設定      |
|`dayMap`     |日程振替・特別ラベル・時間割非表示|
|`className`  |クラス名             |
|`tasks`      |課題データ            |
|`attend`     |欠席カウント・期間設定      |
|`attendWeeks`|前期・後期の授業週数       |

### ICSエクスポート

設定タブ →「カレンダーに追加（ICS）」  
Googleカレンダーなど外部カレンダーアプリに時間割をインポートできます。  
AndroidではShare APIまたはDataURL経由でのダウンロードに対応しています。

## 技術情報

- **フロントエンド**：HTML / CSS / Vanilla JS（フレームワークなし）
- **ストレージ**：localStorage
- **通知**：Web Notifications API + Service Worker
- **PWA**：Web App Manifest + Service Worker（キャッシュ戦略：index.htmlはNetwork First）
- **画像保存**：html2canvas + Web Share API
- **ICSエクスポート**：Share API（text/plain fallback）+ FileReader DataURL fallback
